GOUIDE
======

Использовать ТОЛЬКО классы

Не опираться на элементы, только на классы
.rua-b-user b -> .rua-b-user .rua-b-first-letter

Всем классам внутри блока давать имена начинающиеся с имени этого блока
.rua-b-user .rua-b-first-letter -> .rua-b-user-first-letter


Модификаторы (постфиксы

.rua-b-user_red



.rua-b - блок
.rua-h - holster - обертка для задания отступов
.rua-l - layout - для блоков которые ничего не делают кроме как лейаут
.rua-g - global - глобальные классы (напр. rua-g-hidden)
