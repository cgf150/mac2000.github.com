Install zip:

    
    <code>sudo apt-get install zip
    sudo apt-get install unzip</code>


Unzip:

    
    <code>unzip myarchive.zip</code>


Zip folder:

    
    <code>zip -r myarchive.zip /path/to/my/folder</code>
